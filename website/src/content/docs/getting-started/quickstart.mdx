## Prerequisites

- Python 3.12 or higher
- [UV package manager](https://github.com/astral-sh/uv)
- Docker
- OpenSSL (for generating TLS certificates for local SpiceDB usage)

## Installation

<Steps>
1. **Clone the repository**
   ```bash
   git clone https://github.com/openshift-hyperfleet/kartograph.git
   cd kartograph
   ```

1. **Configure environment**
   
   Default Kartograph configuration files can be found in
   `/env`. 

1. **Run Kartograph**

   Kartograph can quickly be run locally using `docker compose`:
   ```bash
   make run
   ```
   This will first start a Postgres instance with ApacheAGE installed and
   then build and run the Kartograph API.
</Steps>
