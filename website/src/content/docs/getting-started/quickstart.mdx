## Prerequisites

- Python 3.12 or higher
- UV package manager
- Docker (for Apache AGE database)

## Installation

<Steps>
1. **Clone the repository**
   ```bash
   git clone https://github.com/openshift-hyperfleet/kartograph.git
   cd kartograph
   ```

1. **Configure environment**
   
   Default Kartograph configuration files can be found in
   `/env`. 

1. **Run Kartograph**

   Kartograph can quickly be run locally using `docker compose`:
   ```bash
   make run
   ```
   This will first start a Postgres instance with ApacheAGE installed and
   then start the Kartograph API.
</Steps>

## What's Next?

### Using the MCP Server

Kartograph exposes an MCP (Model Context Protocol) server that allows AI agents like Claude to query the knowledge graph using Cypher.

<Steps>
1. **Ensure the API is running**

   The API should be running on `http://localhost:8000` (see Installation above).

2. **Register with Claude Code**

   ```bash
   claude mcp add --transport http kartograph http://localhost:8000/query/mcp
   ```

3. **Query the graph**

   Once registered, the `query_graph` tool will be available in Claude Code
   and you can ask it to use the tool to query the graph.
</Steps>
